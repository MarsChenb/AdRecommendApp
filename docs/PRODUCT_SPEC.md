# AdRecommendApp 产品规格

## 1. 产品定位

AdRecommendApp 是一个面向训练营答辩的广告推荐信息流 Demo。它模拟内容与广告混排推荐场景中的广告信息流体验，重点展示客户端信息流渲染能力、广告卡片动态化能力、AI 内容理解能力和基础统计能力。

产品不追求接入完整商业广告系统，而是优先保证以下目标：

- 功能闭环完整。
- 交互稳定可演示。
- 数据和 AI 链路讲得清楚。
- 工程结构便于继续扩展。

## 2. 目标用户

### 2.1 演示用户

训练营导师、评审和同学。

他们关注：

- App 是否能稳定运行。
- 信息流体验是否完整。
- AI 摘要、标签和搜索是否有实际效果。
- 技术方案是否合理。

### 2.2 终端用户画像

模拟普通移动端用户。

用户诉求：

- 快速浏览感兴趣的广告内容。
- 通过频道、标签和搜索减少无关内容。
- 能进入详情了解商品或服务。
- 能对广告进行点赞、收藏、分享等互动。

## 3. 核心用户路径

### 3.1 信息流浏览

```text
打开 App
→ 默认进入精选频道
→ 浏览单列广告信息流
→ 看到广告图片、标题、价格、AI 摘要和标签
```

验收标准：

- 首屏有内容。
- 列表可流畅滚动。
- 不同卡片样式可区分。

### 3.2 频道切换

```text
点击顶部频道 Tab
→ 切换到电商/本地等频道
→ 信息流内容刷新
```

验收标准：

- 当前频道高亮。
- 切换后展示对应频道广告。
- 切换过程不崩溃。

### 3.3 详情页互动

```text
点击广告卡片
→ 进入广告详情页
→ 查看更完整的图文、AI 推荐理由和标签
→ 点赞/收藏/分享
→ 返回列表
```

验收标准：

- 详情页展示与列表一致的广告内容。
- 点赞和收藏状态返回列表后仍然同步。
- 点击统计增加。
- 返回后列表位置保持。

### 3.4 刷新和加载更多

```text
下拉刷新
→ 当前频道数据重新加载

滑动到底部
→ 加载更多广告
```

验收标准：

- 有加载中状态。
- 有空态或错误态兜底。
- 加载更多不会重复插入异常数据。

### 3.5 标签过滤

```text
点击广告标签，例如“学生党”
→ 信息流只展示相关广告
→ 清除过滤
→ 恢复原信息流
```

验收标准：

- 标签过滤结果符合直觉。
- 当前过滤条件可见。
- 可一键清除。

### 3.6 对话式搜索

```text
输入自然语言需求
例如：我想找适合通勤的降噪耳机
→ 返回匹配广告
→ 可点击进入详情
```

验收标准：

- 支持自然语言输入。
- 能根据标题、描述、AI 摘要和标签匹配。
- 无结果时有明确提示。
- 后续可接真实 AI 做 query 改写或排序。

### 3.7 统计展示

```text
浏览、点击、点赞、收藏、分享
→ 本地记录统计数据
→ 在统计面板展示
```

验收标准：

- 曝光数、点击数、互动数可见。
- 统计口径清楚。
- 数据变化可被演示观察。

## 4. 页面规划

### 4.1 首页 Feed

组成：

- 顶部标题和搜索入口。
- 频道 Tab。
- 当前过滤标签提示。
- 单列广告列表。
- 加载更多状态。
- 统计入口。

### 4.2 广告卡片

共同信息：

- 品牌。
- 标题。
- 图片或视频封面。
- 价格或利益点。
- AI 摘要。
- AI 标签。
- 点赞、收藏、分享按钮。

样式：

- 大图卡片。
- 小图卡片。
- 视频样式卡片。

### 4.3 详情页

组成：

- 返回按钮。
- 大图或视频封面。
- 标题、品牌、价格。
- 详细描述。
- AI 摘要。
- AI 推荐理由。
- 标签、受众、卖点。
- 点赞、收藏、分享。
- 来源链接。

### 4.4 搜索页或搜索面板

组成：

- 输入框。
- 搜索历史或推荐 query。
- 搜索结果列表。
- 空结果提示。

### 4.5 统计面板

组成：

- 总曝光。
- 总点击。
- 点赞数。
- 收藏数。
- 分享数。
- Top 广告列表。

## 5. 数据要求

第一版数据要求：

- 至少 20 条广告。
- 至少 3 个频道。
- 至少 3 种卡片类型。
- 每条广告都有标题、品牌、描述、来源链接。
- 每条广告都有 AI 摘要和 AI 标签。

数据来源应记录在后续 `docs/DATA_SOURCE.md` 中。

## 6. AI 能力要求

AI 至少提供：

- 广告摘要。
- 智能标签。
- 适合人群。
- 核心卖点。

优先采用离线 AI 预处理，输出结构化 JSON，Android 端读取缓存结果。

后续增强：

- 搜索 query 改写。
- 搜索结果重排。
- 详情页推荐理由生成。

## 7. 非目标

第一版不做：

- 完整广告投放系统。
- 用户账号系统。
- 支付和下单。
- 真实商业曝光回传。
- Android 端实时爬虫。
- 列表滑动时逐条实时请求 AI。

## 8. MVP 验收清单

- App 能启动。
- 首页有广告信息流。
- 频道能切换。
- 三种卡片样式都能展示。
- 可以进入详情页并返回。
- 点赞和收藏状态同步。
- 可以刷新和加载更多。
- 可以点击标签过滤。
- 可以输入自然语言搜索。
- 可以查看统计数据。
- AI 摘要和标签真实存在。
- README 和技术文档完整。
